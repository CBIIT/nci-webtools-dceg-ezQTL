Visualization of Quantitative Trait Loci (vQTL) is a web-based tool for integrative GWAS annotation, including eQTL, sQTL, meQTL, miQTL etc. It can also be used to visualize any other genetic association results. In addition, vQTL can perform colocalization analyses between two association results (for example, GWAS and eQTL). For the visualization, vQTL can recalculate linkage disequilibrium (LD) based on a selected variant, which could be useful to prioritize potential causal variants for functional study. Currently, vQTL includes the QTLs-GWAS module. Other modules will be developed soon.

QTLs-GWAS includes the following four sub-modules:

Locus Aligment:

Simultaneously and interactively visualize association p-values and linkage disequilibrium patterns using two locuszoom plots for a locus of interest. A p-value correlation plot between these two locuszoom plots is generated to show the colocalization pattern.

Locus Table:

A sortable table displays all the QTLs with links to different databases (LDlink, GWAS Catalog, gnomAD, etc.)

Locus Colocalization:

Perform colocalization analysis between two association results in a locus using three different methods.

Locus Quantification:

Visualize the quantification data used in the QTL testing in the locus of interest. The quantification data are also used to generate the boxplot for QTLs.
