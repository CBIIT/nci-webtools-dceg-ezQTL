Two locuszoom plots show the association p-value between GWAS and QTLs. If there is no GWAS data, only QTL locuszoom plot will show. Please select the population and LD reference to update the LD pattern. By default, the LD information (R2) will calculated based on the 1000 genome dataset (Phased 3 version 5) but will be replaced by the user’s input.

Pink diamond in the locuszoom plot shows the best (most significant) QTLs based on the selected “Reference Gene”. Red circle indicates the current LD reference variant, which will be simultaneously updated in both GWAS and QTLs locuszoom plots.

The following scatter plot shows the correlation between -log10(GWAS p-value) and -log10(QTLs p-value). The “P-value threshold” can be used to filter QTL and GWAS data before performing correlation analysis. Both person (r) and spearman (rho) correlation coefficients are calculated. For detailed colocalization analysis. Please check the “Locus Colocalization” sub-module.
